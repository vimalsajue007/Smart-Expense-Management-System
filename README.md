# Smart-Expense-Management-System

The Smart Expense Manager is a Python-based application integrated with MySQL that helps users:
•	Track daily expenses
•	Categorize spending
•	Analyze financial habits
•	Generate smart insigh

Project demonstrates real-world application development using:
•	Python
•	MySQL (Database)
•	SQL Queries

Features
•	Add new users
•	Add expense details
•	View user expenses
•	Filter by category
•	Filter by date
•	Calculate total expense
•	Category-wise spending
•	Update expense records
•	Delete expense records
•	Monthly expense report
•	Find highest expense
•	Generate smart insight

Database Setup
•	Create database
•	Create users table
•	Create expenses table
•	Define foreign key
